# 项目介绍
支持插件化、可视化的数据异构中间件，支持的数据异构方式如下  
```
MySQL <——> MySQL（增量、全量）
MySQL <——> Oracle（增量、全量）
Oracle <——> MySQL （增量、全量）
Oracle <——> Oracle（增量、全量）
```
# 使用步骤

## 运行系统

### 下载mykit-data源码。

```bash
git clone https://github.com/sunshinelyz/mykit-data
```

### 使用Maven编译mykit-data

```bash
mvn clean package 
```

### 运行程序

```bash
java -jar mykit-data-starter-xxx.jar
```

### 打开系统连接

```bash
http://localhost:8888
```
